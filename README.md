\# 🍕 Restaurant Order System with Voice Recognition



A complete restaurant ordering system that lets customers speak their orders, with a real‑time kitchen display.



\## ✨ Features

\- 🎤 Voice ordering – say “small pizza”, “coke”, etc.

\- 👨‍🍳 Live kitchen view – orders appear instantly

\- 📱 Mobile‑friendly – works on phones

\- 💾 Automatic bill calculation

\- 🧾 Order confirmation



\## 🚀 How to Run

1\. Clone the repository

2\. Navigate to the `claude-polished` folder

3\. Install dependencies: `pip install -r requirements.txt`

4\. Run the app: `python app.py`

5\. Open `http://127.0.0.1:8080` in your browser
